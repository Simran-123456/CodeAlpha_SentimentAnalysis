# CodeAlpha - Sentiment Analysis Project

## Project Overview
This project is part of the Data Analytics Internship at CodeAlpha.

The project performs sentiment analysis on customer reviews using Natural Language Processing (NLP).

## Technologies Used
- Python
- TextBlob
- Pandas
- Matplotlib

## Features
- Detects Positive, Negative, and Neutral reviews
- Uses NLP for sentiment classification
- Visualizes sentiment distribution using charts
